# airport-operations-optimization
Simulation and optimization of aircraft turnaround operations using Python

# Airport Operations Optimization ✈️

This project models and optimizes airport turnaround operations using **Discrete Event Simulation (SimPy)** and **Linear Programming (PuLP)**.

## Overview
- Simulates aircraft turnaround tasks (cleaning, fueling, boarding, etc.)
- Identifies bottlenecks through simulation metrics
- Tests optimization strategies to reduce average turnaround time
